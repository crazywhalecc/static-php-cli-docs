# static-php-cli-docs
[static-php-cli](https://github.com/crazywhalecc/static-php-cli) Documentation, use VitePress.

```bash
# Install dependencies
yarn

# Dev mode
yarn docs:dev

# Build mode, target: docs/.vitepress/dist/
yarn docs:build
```
